● Platform Features

Everything your lab needs

Built specifically for XR research workflows. No adapting spreadsheets — purpose-built tools for how your lab actually operates.

🔐
Secure Authentication

JWT-based login with bcrypt password hashing. Role-based access for Admins, Researchers, and Viewers — each with tailored permissions.

👥
Participant Registry

Register, search, and track all experiment participants. Consent tracking, experiment history, and soft-delete for full audit compliance.

📅
Smart Scheduling

Conflict-free experiment booking with real-time availability. Automatic email confirmations and equipment reservation in one step.

🔬
API Lookup

Fetch published research from PubMed and OpenAlex. Attach papers directly to experiments for seamless literature tracking.

📋
Activity Logging

Append-only audit log for every action in the system. Know who accessed what, when — exportable for compliance reports.

🛡️
Lab Security

VLAN isolation, UFW firewall, TLS encryption, and rate-limited endpoints. Research data stays in your infrastructure, not the cloud.

Built with a modern stack

Open-source technologies, self-hosted on your institution's infrastructure.

Angular Node.js Express MongoDB Mongoose JWT Auth Nginx Ubuntu VM UFW Firewall VLAN Networking REST API Nodemailer
